# Host Configurations

Per-host NixOS configurations. The inventory file is the single source of truth for which hosts exist — `flake.nix` reads it to generate `nixosConfigurations` and `homeConfigurations`.

## Architecture

```
hosts/
├── _inventory.nix       # Host registry (hostname + stateVersion per host)
├── desktop/             # Desktop PC: gaming, full virt, Mullvad
│   ├── configuration.nix
│   ├── hardware-configuration.nix
│   ├── local-packages.nix
│   └── modules/         # Host-specific overrides (power)
└── thinkpad/            # Laptop: Bluetooth, TLP, NVIDIA dGPU
    ├── configuration.nix
    ├── hardware-configuration.nix
    ├── local-packages.nix
    └── modules/         # Host-specific overrides (boot, nvidia, power, tlp, thermal)
```

## Key Files

| File | Purpose |
|------|---------|
| `_inventory.nix` | Host list — add/remove hosts here, flake.nix reads automatically |
| `*/configuration.nix` | Per-host entry point: imports hardware-config, local-packages, nixos/modules, and ./modules |
| `*/hardware-configuration.nix` | Auto-generated hardware scan (filesystems, kernel modules) |
| `*/local-packages.nix` | Host-specific system packages |
| `*/modules/` | Host-specific module overrides (power, GPU, thermal, TLP) |

## Host Differences

| Feature | desktop | thinkpad |
|---------|---------|----------|
| Gaming (Gamescope) | ✅ | ❌ |
| Virtualization | ✅ | ❌ |
| Mullvad VPN | ✅ | ❌ |
| Bluetooth | ❌ | ✅ |
| TLP power mgmt | ❌ | ✅ |
| NVIDIA dGPU control | ❌ | ✅ (modules/nvidia.nix) |

## Conventions

- To add a host: append entry to `_inventory.nix`, create `hosts/<name>/` with `configuration.nix`
- `configuration.nix` sets `mySystem.*` options to enable/opt-in system modules
- `hardware-configuration.nix` should not be edited manually (generated by `nixos-generate-config`)

## Dependencies

- **Imports**: `../../nixos/modules` (all system modules), `./modules` (host-specific)
- **Imported by**: `flake.nix` via `makeSystem` factory
